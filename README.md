# ComandaFácil 🚀

Sistema inteligente para gerenciamento de pedidos, comandas e estoque.

## ✨ Funcionalidades
- **Painel de Caixa**: Registro de pedidos rápido e intuitivo.
- **Painel de Cozinha**: Acompanhamento de pedidos em tempo real.
- **Painel Admin**: Gestão de estoque e relatórios de vendas.
- **Autenticação**: Segurança integrada com Supabase.

## 🛠️ Tecnologias
- **Next.js 15+** (App Router)
- **Supabase** (Banco de dados e Auth)
- **Tailwind CSS** (Estilização moderna)
- **Lucide React** (Ícones)

## 🚀 Como começar

1. Instale as dependências:
```bash
npm install
```

2. Configure o arquivo `.env.local` com suas chaves do Supabase.

3. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.
